Angular 5 Bootstrap

Demo project using Angular 5 and Bootstrap 4.

This project implements Bootstrap 4 in Angular 5.

As part of creating this project to demonstrate the Bootstrap components advanced routing is also implemented.

Angular 5

Angular 5 is the framework used to build this demo application.

Angular.io
Bootstrap 4

Bootstrap is a front-end component library for web applications.

Bootstrap
ng-bootstrap

ng-bootstrap is a port of Bootstrap to the Angular environment.

Bootstrap is dependent on the jQuery and Popper.js libraries. ng-bootstrap removes these dependencies.
